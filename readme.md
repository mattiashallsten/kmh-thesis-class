# KMH Thesis Class
LaTeX-klass för KMH's formatteringsregler, baserad på [dessa](./guide.txt) regler.

## Installation
Lägg denna mapp där LaTeX letar efter .cls-filer, i mitt fall `$HOME/Library/texmf/tex/`.

- [x] Radavstånd: 1.5.
- [ ] Fotnoter: 9pt
- [x] Marginaler: 3cm från alla håll
